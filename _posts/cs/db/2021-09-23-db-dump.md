## 1. DB 백업

> Shell

```bash
$ mysqldump -u {account-name} -p {db-name} > {file-name}.sql
```

<br>

## 2. DB 복구

> Shell

```bash
$ mysql -u {account-name} -p
$ drop database {db-name}
$ create database {db-name}
$ exit
$ mysql -u {account-name} -p {db-name} < {file-name}.sql
```

* DB를 삭제하고 다시 생성한 뒤 dump 파일로 복구가 되는지 확인한다.
